# Neo4j Cypher Learning Notes

Simple Cypher query files used while learning Neo4j and Cypher.

## Environment

- Neo4j Desktop: 2.1.4
- Neo4j DB version: 2026.03.1

## Purpose

This repo is a personal practice space for:

- CREATE clauses for nodes and relationships
- MATCH clauses for querying graph data
- RETURN clauses for specifying output
- Trying small Cypher exercises

## Files

- `create.cypher` - basic `CREATE` examples and relationship exercises
- `match.cypher` - basic `MATCH` query examples

## How to Use

1. Open your DB in Neo4j Desktop.
2. Open Neo4j Browser for that DB.
3. Copy queries from the `.cypher` files and run them step by step.
4. Use `MATCH` queries to verify what was created.

## Notes

- These queries are intentionally simple and beginner-focused.
- Some scripts may create data repeatedly if run multiple times.
